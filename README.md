## Getting Started with Arcade Game Template

Alter the TEMPLATES array inside gulpfile.js/alza-template.js to match the correct template folder naming.
Change images and cols inside container and|or remove to your need.

## Available Scripts
In the project directory, you can run:

### `gulp`

Runs the default game template from `src/version-hornbach` to the `_temp` folder in the development mode.\
The page will reload if you make edits.

### `gulp build`

Builds the production minified game version from `src/version-hornbach` to the `_build` folder.\
It correctly bundles Flexslider in production mode and optimizes the build for the best performance.

### `gulp banner`

Runs the default banner template from `src/version-banner` to the `_temp_banner` folder in the development mode.\
The page will reload if you make edits.

### `gulp bannerbuild`

Builds the production minified game version from `src/version-banner` to the `_build_banner` folder.\
It correctly bundles Flexslider in production mode and optimizes the build for the best performance.
